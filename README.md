> ## 1.1.1 穷竭搜索<br>
>> DFS：从一个状态开始，不断移动状态，直到无法移动，然后退回到上一状态，继续转移其他状态，不断重复，直到找到最终解。递归函数<br>
>> BFS：从一个状态开始，不断移动状态，总是搜索距离初始状态距离近的状<br>
> ## 1.1.2 优化<br>
>> 特殊状态的枚举:生成可解空间多数采用DFS，但是状态空间较为特殊时可通过全排列解决。<br>
